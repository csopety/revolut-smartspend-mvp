# Revolut SmartSpend MVP

A Python Streamlit MVP for a school assignment. The app will demonstrate pre-purchase grocery basket optimization for Budapest II district using simulated data.

## Planned MVP
- 4 grocery chains
- 3 food categories
- Basket comparison
- Budget fit calculation
- Travel cost and time trade-off
- Store recommendation algorithm
- Simulated savings transfer into a Revolut-style Pocket

## Run
streamlit run app.py

## Test
pytest
